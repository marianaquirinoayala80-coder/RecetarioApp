# Recetario

Este repositorio contiene una aplicación para gestionar recetas de cocina.

## Características

- Añadir, editar y eliminar recetas
- Buscar recetas por nombre o ingredientes
- Interfaz intuitiva y fácil de usar

## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/marianaquirinoayala80-coder/recetario.git
    ```
2. Ingresa al directorio:
    ```bash
    cd recetario
    ```
3. Sigue las instrucciones específicas de instalación en la documentación del proyecto.

## Uso

Ejecuta la aplicación y comienza a gestionar tus recetas favoritas.
